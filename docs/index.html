関数を作ってみよう R-Source::関数事始

標本分散を計算する関数を作る

Rの var() 関数は，不偏分散を計算する関数なので， 標本分散を計算する関数を定義してみる: 

(varp <- function (x) {
  var(x)*(length(x)-1)/length(x)
})

function (x) {
  var(x)*(length(x)-1)/length(x)
}

(x <- c(10, 13, 8, 15, 8))
var(x)
varp(x)
